# Mahjong-score-tracker
麻雀の戦績を 簡単に計算・登録・共有できる Flask 製の Web アプリ
## 新機能: 称号/バッジシステム

プレイヤーは条件を満たすことで称号を獲得し、プロフィール上に3つまで装備できます。

### 称号の種類と獲得条件

- **初心者**: ゲーム開始時に自動取得
- **5戦突破**: 5戦以上の対戦
- **10戦達成**: 10戦以上の対戦
- **安定王**: 5戦以上で平均着順 2.5以下
- **ラスター**: 5戦以上でラス回避率 75%以上
- **チャンピオン**: 10戦以上で平均着順 2.0以下、1位3回以上、ラス回避率 80%以上

### 使用方法

1. プロフィールページで「装備中の称号」セクションを確認
2. スロット(1-3)をクリックしてモーダルを開く
3. 獲得済みの称号から選択、または「外す」で削除
4. 選択した称号がプロフィールに表示されます

### 技術実装

**モデル**:
- `Title`: 称号定義(name, badge_filename, 取得条件fields)
- `Player`: 3つの称号スロット(active_title_1/2/3_id)

**API**:
- `POST /api/player/<id>/title/update`: 称号装備状態を更新

**フロントエンド**:
- プロフィール上に3つのバッジスロット表示
- モーダルで獲得済み称号から選択可能
- 空きスロットは黒い円で表示
